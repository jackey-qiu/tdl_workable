The xray Data analysis Library (Tdl) is a python based project that assembles a number of x-ray data analysis routines into a single package. This project is still in a very early stage of development.  

For more information see:
doc/INSTALL.txt
doc/LICENSE.txt

The primary documentation pages are currently at: 
http://cars9.uchicago.edu/ifeffit/tdl

These will be moved over to the github wiki soon:
https://github.com/xraypy/tdl/wiki
